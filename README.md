# Network-Automation with Ansible
---
Collection of ansible playbooks for configuring network devices. The labs will be configured with EvE-NG and will mostly consits of Cisco IOS. 

---

# WHAT IS EVE-NG?
- EvE-NG is a clientless multivendor network emulation software that gives security and network professionals a safe place to tinker without influence of corporate security policies. It is a completly isolated environment. More information on EvE-NG can be found here https://www.eve-ng.net/
